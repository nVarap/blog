Example One 

lst = [1, 100, 123, 214, 123, 931]
new_lst = []
for num in lst:
    if num % 4 == 2:
        new_lst.append("err")
print(new_lst)

['err']

Explanation

Everything is sequencing
for num in lst is iteration, as it iterates over the set list.
if num % 4 == 2 is selection, as it selects for a specific value

    c = a+b
    d = a+b+c
    e = (a+b+c+d)/(d-b)
    display(e)

Answer:

c = 3 d = 6 e = 1+2+3+6/(6-2) = 12/4 = 3
